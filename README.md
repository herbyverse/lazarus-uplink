# Lazarus Uplink

*A soft terminal game about fragment alignment, corruption, and emotional stability.*

---

## Overview 

Lazarus Uplink is a narrative-driven game where the player acts as the **Keeper**, guiding inner fragments through cycles of recorevy, destabilization, and integration.

---

## Core Features

- Fragment personalities (The Watchful One, The Hearthkeeper, and the Performer)
- Dialouge thresholds and Keeper decision-making
- Dynamic consequences based on fragment state
